
#### Usage

1. Installation 

	```bash
	apt-get install theharvester
	```


2. Options


	```shell
	-d        Domaine de recherche
	-l        Limite le nombre de résultat
	-b        Moteur de recherche
	```



3. Search for a domain name

	```shell
	theharvester -d domain.fr -b all
	```
